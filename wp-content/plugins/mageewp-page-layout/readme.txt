== Description ==

Mageewp Page Layout, accompanied with different kinds of page layout styles, makes page building and editing a much easier thing.Mageewp Page Layout also comes with pre-set modules like Banner, promo, service and team etc., which will help users build their site page layout from blank canvas whithin few minutes. More page layout styles you can expect from our new updates.

== Installation ==

Unzip plugin files and upload them under your '/wp-content/plugins/' directory.

Resulted names will be: './wp-content/plugins/mageewp-page-layout/*'

Activate plugin at "Plugins" administration page.
